A Python-based Voice Assistant that listens to your voice commands and performs useful tasks like telling jokes, opening applications, playing songs, and more—just like your own mini Jarvis!

---


## 🚀 Project Overview

This project demonstrates how to build a basic, interactive voice assistant using Python and popular speech libraries. It leverages offline text-to-speech synthesis and online speech recognition to deliver a seamless hands-free experience.

### 🎯 Ideal For:
* **Python beginners** looking to work with real-world automation 🐍
* Developers learning **Speech Recognition & NLP** fundamentals 🎧
* Anyone building foundational blocks for **AI-based projects** 🤖

---

## ✨ Features

* ✅ **Voice Recognition:** Live audio capture via microphone using advanced speech recognition algorithms.
* ✅ **Text-to-Speech (TTS):** Natural-sounding audio responses powered by an offline synthesis engine.
* ✅ **Programming Humor:** Fetches and delivers clever programming jokes on demand.
* ✅ **Application Automation:** Instantly launches system applications like WhatsApp, web browsers, and more.
* ✅ **Media Control:** Plays your favorite music tracks via simple voice commands.
* ✅ **Beginner-Friendly:** Highly modular, readable, and well-commented codebase.

---

## 🛠️ Technologies & Packages

### Built With
* **Python 3.10+**

### Required Dependencies
* `SpeechRecognition` — For converting spoken audio into text strings.
* `pyttsx3` — Offline text-to-speech conversion library.
* `pyjokes` — One-liner jokes for programmers.
* `pyaudio` — Required for cross-platform microphone audio input capture.

### 📦 Installation

Install all the required dependencies using `pip`:

```bash
pip install SpeechRecognition pyttsx3 pyjokes pyaudio
